# MyDay

**Daily routine website and app**

This project, *MyDay*, is a simple and interactive daily routine planner built with HTML, CSS, and JavaScript. It helps users organize their daily tasks and manage their time efficiently.

## Features

- Add, edit, and delete daily tasks
- Responsive design for mobile and desktop
- User-friendly interface

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Vibes coding (your custom approach)

## How to Use

1. Open `index.html` in your browser to start using the app.
2. Add your daily tasks and manage your routine easily.
